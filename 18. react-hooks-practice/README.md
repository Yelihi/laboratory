# React Hooks Practice

React + Vite + React Router + TailwindCSS + TypeScript 기반의 React Hooks 학습 프로젝트입니다.

## 기술 스택

- **React 18**: UI 라이브러리
- **Vite**: 빌드 도구
- **React Router DOM**: 라우팅
- **TailwindCSS**: 스타일링
- **TypeScript**: 타입 안정성

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 프로젝트 구조

```
src/
├── pages/          # 페이지 컴포넌트
│   ├── Home.tsx
│   └── About.tsx
├── components/     # 공통 컴포넌트
├── hooks/         # 커스텀 훅
├── App.tsx        # 메인 앱 컴포넌트
├── main.tsx       # 진입점
└── index.css      # 전역 스타일
```

## 라우팅

- `/` - Home 페이지
- `/about` - About 페이지
